# Support

This repository is the umbrella project hub for Threadbase.

For setup help or implementation-specific issues, please start with the relevant component repository.

## Before opening an issue

Please include:

- which Threadbase repo/component you are using,
- OS and version,
- installation method,
- relevant command output,
- screenshots or logs if useful,
- whether the issue happens locally, remotely, or through a tunnel.

Do not include API keys, pairing tokens, private prompts, or private session content.
