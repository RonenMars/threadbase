# Code of Conduct

Threadbase aims to be welcoming, respectful, and constructive.

## Expected behavior

- Be respectful and kind.
- Assume good intent.
- Give constructive, technical feedback.
- Respect different experience levels.
- Protect private information and secrets.

## Unacceptable behavior

- Harassment or discrimination.
- Threats or intimidation.
- Personal attacks.
- Repeated off-topic disruption.
- Publishing private information without permission.
