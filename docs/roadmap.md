# Roadmap and Project Status

Threadbase is under active development.

## Current status

| Area | Status |
|---|---|
| Mobile app | Beta |
| Streamer | Active |
| Scanner | Active |
| VS Code / IntelliJ / Electron clients | Active / evolving |
| Menubar | Lightweight utility |
| Multi-agent orchestration | Experimental |

## Near-term focus

- improve onboarding and pairing,
- harden streamer security and connectivity,
- clarify and expand provider support,
- improve cross-client session experience,
- strengthen local history search.

## Longer-term ideas

- richer provider adapters,
- better session replay,
- optional sync and team workflows,
- orchestration dashboards.

Future features should be explicitly marked as roadmap or experimental rather than current functionality.
